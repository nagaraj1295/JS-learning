# JS-learning

## Overview

This repository is a small JavaScript learning project containing basic examples and concepts for beginners.

## Files

- `index.html`: Web page that loads the example scripts.
- `variableScript.js`: Demonstrates variable declaration, assignment, and console output.
- `dataTypeNum.js`: Demonstrates number data types and simple arithmetic operations.
- `README.md`: Project documentation and usage guide.

## Project goals

- Learn basic JavaScript syntax and structure.
- Practice variables, data types, and simple operations.
- Build a small interactive example in the browser.

## Usage

1. Open `index.html` in a web browser.
2. Open the browser console to view the output from the script files.
3. Edit the scripts in this folder to test new concepts.

## Notes

- `let` is a JavaScript keyword used to declare variables with block scope.
- This repo is intended for practice and experimentation with core JavaScript fundamentals.
